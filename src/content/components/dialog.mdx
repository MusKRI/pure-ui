## Anatomy

Import the component and assemble its parts:

```tsx
import { Dialog } from "@/components/ui/dialog";

<Dialog>
  <DialogTrigger />
  <DialogPortal>
    <DialogBackdrop />
    <DialogPopup>
      <DialogTitle />
      <DialogDescription />
      <DialogClose />
    </DialogPopup>
  </DialogPortal>
</Dialog>;
```

## Usage

### Non Dismissible

By default, the dialog is dismissible. You can make it non-dismissible by passing `dismissible={false}` to the `Dialog` component. This is useful when you want to prevent the user from closing the dialog by clicking outside of it or by pressing the escape key.

<ComponentShowcase name="dialog-non-dismissible-demo" />

### Non Modal

The `modal` property controls how users can interact with the page when the dialog is open.

**`modal={true}`** (default) -
Focus is trapped, scrolling is locked, and outside interactions are disabled.

**`modal={false}`** -
Users can interact with both the dialog and the rest of the page freely.

**`modal="trap-focus"`** -
Focus is trapped for keyboard navigation, but scrolling and outside pointer interactions remain enabled.
