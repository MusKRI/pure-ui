## Anatomy

```tsx
import { Tooltip, TooltipTrigger, TooltipPopup } from "@/components/ui/tooltip";

<Tooltip>
  <TooltipTrigger />
  <TooltipPopup>// Your tooltip content</TooltipPopup>
</Tooltip>;
```

## Usage

### With Arrow

You can pass `showArrow` as true to `TooltipPopup` to show arrow on tooltip. By default, it is false.

<ComponentShowcase name="tooltip-with-arrow-demo" />

### Sides

You can pass `side` to `TooltipPopup` to set the side of the tooltip. By default, it is `top`.

<ComponentShowcase name="tooltip-sides-demo" />

### Offset

You can pass `sideOffset` to `TooltipPopup` to change the offset of the tooltip. By default, it is 4.

<ComponentShowcase name="tooltip-offset-demo" />

<ComponentShowcase name="tooltip-controlled-demo" />

### Delay

You can control the `open` and `close` delay of the tooltip with `delay` and `closeDelay` props on `Tooltip` component.
