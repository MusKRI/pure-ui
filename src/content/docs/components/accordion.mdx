## Anatomy

```tsx
import {
  Accordion,
  AccordionItem,
  AccordionTrigger,
  AccordionPanel,
} from "@/components/ui/accordion";

<Accordion>
  <AccordionItem>
    <AccordionHeader>
      <AccordionTrigger />
    </AccordionHeader>
    <AccordionPanel />
  </AccordionItem>
</Accordion>;
```

## Variants

You can use the `variant` prop to change the variant of the accordion.

```tsx noCopy highlight={2}
<Accordion variant="default">...rest content...</Accordion>
```
