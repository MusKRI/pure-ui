## Usage

```tsx
import { Calendar } from "@/components/ui/calendar";
```

```tsx
const [date, setDate] = React.useState<Date | undefined>(new Date());

return (
  <Calendar
    mode="single"
    selected={date}
    onSelect={setDate}
    className="rounded-lg border"
  />
);
```

See the [React DayPicker](https://react-day-picker.js.org) documentation for more information.

## Animate

You can animate the calendar by adding the `animate` prop.

```tsx add=2 noCopy
<Calendar
  animate
  mode="single"
/>
```

### Multiple Months

You can display multiple months by adding the `numberOfMonths` prop. For example, to display 2 months side by side, you can use the following code:

```tsx add=2 noCopy
<Calendar
  multipleMonths={2}
  mode="single"
/>
```

<ComponentShowcase name="calendar-multiple-months-demo" />


### Single Selection

You can use `mode="single"` to allow a single date to be selected.

```tsx add=2 noCopy
<Calendar
  mode="single"
/>
```

<ComponentShowcase name="calendar-simple-demo" />

### Multiple Selection

You can use `mode="multiple"` to allow multiple dates to be selected.
You can also use the `max` prop to limit the number of dates that can be selected.

```tsx add=2 noCopy
<Calendar
  mode="multiple"
/>
```

<ComponentShowcase name="calendar-multiple-selection-demo" />

### Range Selection

You can use `mode="range"` to allow a range of dates to be selected.

```tsx add=2 noCopy

  mode="range"
```
